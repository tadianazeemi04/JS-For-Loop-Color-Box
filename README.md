# JS For Loop Color Box

This project generates a series of colored paragraphs using nested `for` loops in **JavaScript**. The paragraphs display different RGB color combinations as their background, incrementing in steps of 10 for each RGB value.

## Features:
- Dynamically generates paragraphs with unique background colors.
- Each paragraph shows the RGB value of its background color.

## How to Run the Project:

1. **Clone the Repository**:
   - Clone the project to your local machine using the following command:
     ```bash
     git clone https://github.com/tadianazeemi04/js-for-loop-color-box.git
     ```

   Or download the ZIP file and extract it.

2. **Extract Files** (if downloaded as a ZIP):
   - After extraction, you should see the following structure:
     ```plaintext
     js-for-loop-color-box/
     ├── index.html        # Main HTML file
     ├── README.md         # Project details and instructions
     ```

3. **Open the `index.html` File**:
   - Simply double-click `index.html` or open it in your web browser.

4. **See the Colorful Output**:
   - The webpage will dynamically generate colored boxes with corresponding RGB values.

## Technologies Used:
- HTML5
- CSS3
- JavaScript

## Project Overview:
- This project demonstrates how to use nested `for` loops in JavaScript to create RGB color combinations and apply them dynamically to HTML elements.
